JDBC란?
===============
자바 애플리케이션에서 DBMS 종류에 상관없이 일관성 있게 연결해주는 기능


드라이버
-----------------------------
DBMS 개발사가 구현한 클래스의 집합

자바 애플리케이션에서 여러 종류의 DBMS와 연동과정
========================

#### 자바프로그램->JDBC 인터페이스 -> Oracle드라이버 -> Oracle

#### 자바프로그램->JDBC 인터페이스 -> DB2드라이버    -> DB2
                            
#### 자바프로그램->JDBC 인터페이스 -> mySQL드라이버  -> mySQL
